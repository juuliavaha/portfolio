PORTFOLIO WEBSITE

This is my personal portfolio website, designed and developed to showcase my projects, skills, and experience in web development.

The website was built entirely from scratch without using any pre-made templates or themes. All structure, styling, and functionality were implemented manually using HTML, CSS, and JavaScript.

Features

- Fully custom design (no templates used)
- Responsive layout for different screen sizes
- Clean and minimal user interface
- Interactive elements implemented with JavaScript
- Project showcase section
- Contact and information sections

  
Technologies Used

- HTML
- CSS
- JavaScript


Purpose

The goal of this project was to create a personal platform to present my work and to practice building a complete website from design to implementation.


If you want to explore the code structure, you can browse the repository files.
