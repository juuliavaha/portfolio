Portfolio Website

My Portfolio Project


I wanted to create my own portfolio website that serves both as a showcase of my skills and as a demonstration of my web design and development abilities. The website was fully designed and implemented from scratch without using any ready-made themes or templates, and all code was written from a blank project.

The site was primarily designed for large screens, but I also focused on responsiveness. The goal was to ensure that the user experience remains smooth and visually appealing on mobile devices as well.

The website was built using HTML, CSS and JavaScript to implement interactivity and layout design.
