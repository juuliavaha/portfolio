Foodie Planner

Personal Calorie Tracking Application


Foodie Planner Foodie Planner is a web application built with React and TypeScript that allows users to track their daily energy expenditure and manage a food diary.

The purpose of the application is to help users monitor their daily energy consumption and compare it with the calories from consumed meals. Users can calculate their total daily energy expenditure and build a food diary from predefined meals.


Meal management is implemented using a component-based structure. Users can add and remove meals, and each meal has a unique identifier (UUID) used for management purposes. The application automatically calculates the total calories in the food diary whenever meals are added or removed. All data is stored in localStorage, so the diary persists even after refreshing the browser without a separate backend service.

The user interface is built with Material UI components and includes features such as Select inputs, a navigation Drawer, and a responsive list view. The application also includes a separate recipe view where users can check ingredients and preparation instructions for each meal.

Key Features

  • Energy expenditure calculator (age, height, weight, gender, and activity level using the BMR formula and an activity multiplier)
  • Food diary with the ability to add and remove meals
  • Meal list and recipe page displaying the calorie content of each meal
  • Data stored locally in the browser using localStorage
  • Modern user interface built with Material UI components, including menus, text fields, buttons, and a navigation drawer

Technologies Used

  • React (TypeScript)
  • Material UI
  • localStorage
  • UUID
  • React state management
